Module 1: Navigation du Portail

Apprenez à naviguer dans le portail requérant

Leçon 1.1: Interface du Portail

Accès au Portail Requérant

Le portail requérant SAMA PROMIS vous permet de:

  • Consulter les opportunités de financement
  • Soumettre des demandes et propositions
  • Suivre l'état de vos dossiers
  • Communiquer avec les gestionnaires
  • Accéder à vos documents

Structure du Portail

Le portail est organisé en sections principales:

  • Tableau de bord: Vue d'ensemble de vos activités
  • Opportunités: Appels à propositions disponibles
  • Mes dossiers: Vos demandes et propositions
  • Messages: Communications avec l'administration
  • Mon profil: Informations de votre organisation

Première Connexion

Lors de votre première connexion:

  1. Complétez votre profil utilisateur
  2. Ajoutez les informations de votre organisation
  3. Téléchargez les documents requis
  4. Configurez vos préférences de notification
Points Clés
  • Le portail est accessible 24/7
  • Toutes les interactions sont tracées
  • Les notifications vous tiennent informé
  • L'interface est intuitive et responsive
