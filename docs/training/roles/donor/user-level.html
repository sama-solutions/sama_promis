Module 1: Portail Public

Consultez les projets financés via le portail de transparence

Leçon 1.1: Accès au Portail Public

Qu'est-ce que le Portail Public?

Le portail public de SAMA PROMIS permet aux bailleurs de fonds de:

  • Consulter tous les projets financés
  • Suivre l'avancement en temps réel
  • Accéder aux rapports de conformité
  • Visualiser les indicateurs de performance
  • Exporter les données pour analyse

Accéder au Portail

Pour accéder au portail public:

  1. Visitez l'URL du portail public SAMA PROMIS
  2. Utilisez les filtres pour trouver vos projets
  3. Consultez les détails de chaque projet
  4. Téléchargez les rapports disponibles
Points Clés
  • Le portail public offre une transparence totale
  • Les données sont mises à jour en temps réel
  • Filtres avancés pour recherche ciblée
  • Export des données en plusieurs formats
