Module 1: Gestion de Projets

Créez et gérez des projets dans SAMA PROMIS

Leçon 1.1: Création d'un Projet

Accéder au Module Projets

Pour créer un nouveau projet dans SAMA PROMIS:

  1. Connectez-vous à SAMA PROMIS
  2. Allez dans le menu Projets
  3. Cliquez sur Créer

Informations de Base du Projet

Remplissez les champs obligatoires:

  • Nom du Projet: Titre descriptif du projet
  • Code: Identifiant unique (ex: PROJ-2024-001)
  • Type de Projet: Sélectionnez parmi les types prédéfinis
  • Ministère/Organisation: Entité responsable
  • Dates: Date de début et date de fin prévue
  • Budget Total: Montant global du projet

Description et Objectifs

Documentez clairement:

  • Description détaillée du projet
  • Objectifs généraux et spécifiques
  • Résultats attendus
  • Bénéficiaires cibles
Exercice Pratique

Objectif: Créer votre premier projet

  1. Créez un projet de test nommé "Formation SAMA PROMIS"
  2. Définissez un budget de 100 000 000 FCFA
  3. Ajoutez une description et des objectifs
  4. Sauvegardez le projet
Points Clés
  • Chaque projet doit avoir un code unique
  • Le type de projet détermine les workflows disponibles
  • Une description claire facilite le suivi
  • Les dates sont essentielles pour la planification
