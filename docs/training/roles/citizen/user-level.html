Module 1: Portail Public

Découvrez le portail public de transparence

Leçon 1.1: Introduction au Portail Public

Qu'est-ce que le Portail Public SAMA PROMIS?

Le portail public de transparence permet à tous les citoyens de:

  • Consulter tous les projets publics
  • Voir les budgets et financements
  • Suivre l'avancement des projets
  • Accéder aux marchés publics
  • Consulter les rapports de conformité
  • Exporter les données pour analyse

Accès au Portail

Le portail public est accessible sans inscription:

  1. Visitez l'URL du portail public SAMA PROMIS
  2. Aucune connexion n'est requise
  3. Toutes les données sont publiques
  4. Navigation intuitive et responsive

Sections du Portail

Le portail comprend les sections suivantes:

  • Projets: Liste de tous les projets publics
  • Bailleurs: Informations sur les bailleurs de fonds
  • Marchés: Appels d'offres et attributions
  • Statistiques: Tableaux de bord et indicateurs
  • Carte: Visualisation géographique des projets
Exercice Pratique

Objectif: Explorer le portail public

  1. Accédez au portail public SAMA PROMIS
  2. Recherchez un projet dans votre région
  3. Consultez les détails du projet
  4. Explorez les statistiques globales
Points Clés
  • Le portail est accessible à tous sans inscription
  • Toutes les données sont publiques et transparentes
  • Les informations sont mises à jour en temps réel
  • L'interface est simple et intuitive

Leçon 1.2: Navigation et Recherche

Recherche de Projets

Utilisez les filtres pour trouver des projets:

  • Par région: Filtrez par région administrative
  • Par secteur: Santé, éducation, infrastructure, etc.
  • Par bailleur: Filtrez par source de financement
  • Par statut: En cours, terminé, planifié
  • Par budget: Plage de budget

Détails d'un Projet

Pour chaque projet, vous pouvez consulter:

  • Description et objectifs
  • Budget total et sources de financement
  • Dates de début et de fin
  • Ministère/organisation responsable
  • Marchés associés
  • Indicateurs de performance
  • Documents publics
Points Clés
  • Les filtres permettent une recherche ciblée
  • Chaque projet a une page détaillée
  • Les données sont vérifiées et officielles
  • La carte interactive facilite la localisation
