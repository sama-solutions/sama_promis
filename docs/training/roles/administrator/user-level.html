Module 1: Introduction et Installation

Découvrez SAMA PROMIS et apprenez à l'installer sur Odoo 18 CE

Leçon 1.1: Vue d'ensemble de SAMA PROMIS

Qu'est-ce que SAMA PROMIS?

SAMA PROMIS (Project Management Information System) est un système complet de gestion de projets développé pour le gouvernement sénégalais. Il permet de gérer l'ensemble du cycle de vie des projets, de la planification à l'exécution, en passant par le suivi et l'évaluation.

Architecture Micromodules

SAMA PROMIS est construit sur une architecture modulaire comprenant:

  • Core: Fonctionnalités de base (projets, contrats, partenaires)
  • Projects: Gestion avancée des projets et workflows
  • Public Portal: Portail de transparence publique

Fonctionnalités Clés (Phases 1-4)

Phase 1

Gestion multi-sources de financement avec support des devises multiples

Phase 2

Plans de passation de marchés et gestion des appels d'offres

Phase 3

Gestion de la conformité avec profils par bailleur (BM, CE, AFD)

Phase 4

Portail public de transparence et portail requérant

Prérequis Système

Python 3.11+
PostgreSQL 13+
Odoo 18 Community Edition
4GB RAM minimum
10GB espace disque
Points Clés
  • SAMA PROMIS est un système complet de gestion de projets
  • Architecture modulaire pour flexibilité et évolutivité
  • 4 phases complétées: financement, passation, conformité, transparence
  • Compatible avec Odoo 18 CE

Leçon 1.2: Installation sur Odoo 18 CE

Étape 1: Prérequis

Assurez-vous que votre système dispose de:

# Vérifier la version Python
python3 --version

# Vérifier PostgreSQL
psql --version

Étape 2: Installation d'Odoo 18 CE

# Cloner Odoo 18
git clone https://github.com/odoo/odoo.git --depth 1 --branch 18.0

# Installer les dépendances
pip3 install -r odoo/requirements.txt

Étape 3: Installation de SAMA PROMIS

# Cloner le module SAMA PROMIS
cd odoo/addons
git clone <repository_url> sama_promis

# Redémarrer Odoo
./odoo-bin -c odoo.conf --update sama_promis

Étape 4: Vérification

  1. Connectez-vous à Odoo (http://localhost:8069)
  2. Allez dans Apps
  3. Recherchez "SAMA PROMIS"
  4. Cliquez sur "Installer"
Exercice Pratique

Objectif: Installer SAMA PROMIS sur votre environnement de test

  1. Préparez un environnement Odoo 18 CE
  2. Installez le module SAMA PROMIS
  3. Vérifiez que tous les micromodules sont chargés
  4. Prenez une capture d'écran du tableau de bord
Problèmes Courants
  • Erreur de version Python: Utilisez Python 3.11 ou supérieur
  • Dépendances manquantes: Installez toutes les dépendances avec pip
  • Erreur PostgreSQL: Vérifiez que le service est démarré

Leçon 1.3: Premier Démarrage

Accès au Système

Après l'installation, accédez à SAMA PROMIS via:

  • URL: http://localhost:8069
  • Utilisateur: admin
  • Mot de passe: (défini lors de l'installation)

Assistant de Configuration Initiale

Au premier démarrage, l'assistant vous guidera pour:

  1. Configurer les informations de l'organisation
  2. Définir la devise principale
  3. Activer les micromodules nécessaires
  4. Créer les premiers utilisateurs

Vue d'ensemble du Tableau de Bord

Le tableau de bord administrateur affiche:

  • Nombre total de projets
  • Utilisateurs actifs
  • Tâches de conformité en attente
  • Plans de passation actifs
  • Statistiques du portail public
Points Clés
  • L'installation nécessite Python 3.11+ et PostgreSQL 13+
  • SAMA PROMIS s'installe comme un module Odoo standard
  • L'assistant de configuration facilite le premier démarrage
  • Le tableau de bord offre une vue d'ensemble complète
