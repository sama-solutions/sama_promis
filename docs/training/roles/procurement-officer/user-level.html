Module 1: Plans de Passation

Créez et gérez les plans de passation de marchés

Leçon 1.1: Création d'un Plan de Passation

Qu'est-ce qu'un Plan de Passation?

Le plan de passation de marchés est un document stratégique qui:

  • Liste tous les marchés prévus pour un projet
  • Définit les méthodes de passation
  • Établit le calendrier de passation
  • Estime les budgets par marché
  • Identifie les sources de financement

Créer un Plan de Passation

Pour créer un plan de passation dans SAMA PROMIS:

  1. Allez dans Passation de Marchés > Plans de Passation
  2. Cliquez sur Créer
  3. Sélectionnez le projet concerné
  4. Définissez l'année du plan
  5. Ajoutez les lignes de marché

Informations par Ligne de Marché

Pour chaque marché, renseignez:

  • Description: Nature du marché
  • Méthode: AOI, AON, Consultation, Gré à gré
  • Type: Travaux, Fournitures, Services
  • Budget estimé: Montant prévisionnel
  • Dates clés: Publication, ouverture, attribution
  • Source de financement: Bailleur concerné
Exercice Pratique

Objectif: Créer un plan de passation simple

  1. Créez un plan de passation pour un projet test
  2. Ajoutez 3 lignes de marché différentes
  3. Définissez les méthodes et budgets
  4. Validez le plan
Points Clés
  • Le plan de passation est obligatoire pour chaque projet
  • Il doit être approuvé par le bailleur
  • Les méthodes doivent respecter les seuils réglementaires
  • Le plan peut être révisé en cours d'année
